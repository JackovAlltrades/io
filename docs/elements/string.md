### `IoString` `<io-string>` ###

Input element for `String` data type.

#### Properties ####

| Property | Type | Description | Default |
|:--------:|:----:|:----------:|:-------:|
| **`value`** | String | Value. | `''` |

#### Events ####

| Event | Description | Detail |
|:--------:|:----:|:----------:|
| **`'value-set'`** | Value set by user action | `value`, `oldValue` |
